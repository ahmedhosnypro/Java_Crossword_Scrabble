Description

A crossword puzzle consists of a square grid and blanks where crossword fans write letters vertically and horizontally.

In this stage, draw a grid of 15 by 15 and fill each cell with #. Starting from the tile 4-3 (ROW-COL) clear space for the word CROSSWORD across and SCRABBLE from 2-A down.

Objectives
  • Draw a grid of 15 by 15 and fill each cell with # except for cells saved for CROSSWORD and SCRABBLE (see Examples);
  • Label the rows and columns from 1 to F;
  • Change the menu so that it includes the following items: A. Place B. Pick C. Exit
  • Allow the user to place CROSSWORD and SCRABBLE in the available spaces. Do this by first letting them pick the letters DSCBLWORRBAOSE from the bag (for this stage, only those letters should be picked - not random letters and not just 7 at a time) with the Pick command and then place the words with the Place command.
  • Write code to choose the above letters from the bag and put them in the user's tray. To add the words, write Place CROSSWORD 4 3 ACR andPlace SCRABBLE 2 A DWN. ACR denotes "across" and DWN denotes "down"; Letter selection should be case-insensitive. Once placed on the board, all letters should be shown as uppercase.
  • If users select the Place command with an empty tray, print Your tray is empty.
  • If the user places a letter that's not in the tray (case-insensitive), display Invalid letter. Letter X is not in the tray.
  • If the user tries to place at an invalid position (cannot be parsed as hexadecimal), print Invalid command. Use 'Place Word row(Hexadecimal) column(Hexadecimal) DIRECTION'.
  • If the Place command points to a cell with # or an out-of-range cell (<1 or >F), display Invalid position. Position out of range. If the direction is not ACR/ACROSS or DWN/DOWN, print Invalid direction. You should check for valid position before validating direction.
  • If the placed word is not correct, print Incorrect word, please try again.
  • If the user attempts to place a word too long to fit in the available space at their specified position, display Incorrect word. The word is too long.
  • For any 'invalid' error message, the message should appear after the board is reprinted, before the tray/bag lines. (ie., the 17th line of output)
