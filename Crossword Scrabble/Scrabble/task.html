Description

Shuffle the tiles and allow users to pick seven of them to form words from the chosen letter tiles. As users pick tiles, reduce the number of them in the bag and increase the number of tiles in the tray.

Objectives

Build on the program you created in stage 1, your program should do the following;

  • Your program should print the tiles in the tray (if there are tiles on the tray), the bag's content, and prompt users with a menu: 
    A. Shuffle

B. Pick

C. Exit
  • When Shuffle or A is selected, the program should shuffle the tiles in the bag and print out the bag's content in one line. If the bag does not contain any tiles, print The bag is empty. as your first line of output.
  • When Pick or B is selected, the program should randomly pick seven tiles from the bag (or however many remain, if less than seven) and add them to the end of the tray. Display the chosen tiles in a single line; in the second line, print the bag's content without shuffling. If there are no more tiles to choose from, print No more tiles to pick. as your first line of output.
  • When Exit or C is selected, display Bye. and terminate the program. Your program should run continuously until terminated by this command.
  • If the command is not valid, display Invalid Command. as your first line of output and print the menu. Any command should be case-insensitive.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Note that * represents Blank/Wild tiles.
