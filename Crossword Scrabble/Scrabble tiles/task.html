Description

Our scrabble has 100 tiles with alphabet letters with different tile distributions. Write a program that reads the distribution of the tiles from the file and puts them in a bag in alphabetic order. The file name of the distribution file should be passed as a command-line argument, for example:

distribution.txt

as the first and only argument. (Note: the actual file could be named anything, you just have to read its name from the argument)

Objectives
  • Read the file where each line contains a letter followed by the number of tiles of that letter in the bag.
  • Put all the letters and display them on a single line. The file name for the distribution file should be passed as a command-line argument.
Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Note that * represents Blank/Wild tile

Example 1:

File content:

A 6

B 3

C 2

...

Output:

Welcome to Crossword Scrabble

Bag: AAAAAABBBCCDDDDEEEEEEEEEEEEFFGGGHHIIIIIIIIIJKLLLLMMMMNNNNNNOOOOOOOOPPQRRRRRRSSSSTTTTTTUUUUVVWWXYYZ**
